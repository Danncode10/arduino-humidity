================================================================================
                         WIRE COLOR GUIDE
================================================================================

   RED    ──────  5V power          (Arduino 5V  →  DHT11 VCC)
   BLACK  ──────  Ground            (Arduino GND →  DHT11 GND)
   YELLOW ──────  Data signal       (Arduino D2  →  DHT11 DATA)
   ORANGE ──────  Pull-up resistor  (5V rail → 10kΩ → DATA line)

================================================================================
                         COMPONENTS LIST
================================================================================

   [1]  Arduino UNO R3
   [2]  DHT11 Temperature & Humidity Sensor (4-pin module or bare)
   [3]  10kΩ Resistor  (Brown-Black-Orange bands)  ← REQUIRED for stable data
   [4]  Half-size Breadboard
   [5]  Jumper Wires: 1x RED, 1x BLACK, 1x YELLOW, 1x ORANGE
